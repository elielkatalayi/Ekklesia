## 📂 Détail de ce que contient chaque fichier

### **1. Configuration (config/)**

| Fichier | Contenu |
|---------|---------|
| `vite.config.js` | Configuration Vite (port, proxy, plugins) |
| `tailwind.config.js` | Thème Tailwind (couleurs, fonts, extensions) |
| `postcss.config.js` | Configuration PostCSS |
| `.env` | Variables d'environnement (API_URL, etc.) |

### **2. API (api/)**

| Fichier | Contenu |
|---------|---------|
| `axios.config.js` | Configuration Axios (baseURL, interceptors) |
| `auth.api.js` | Fonctions d'appel (login, verify, chooseChurch) |
| `church.api.js` | Fonctions d'appel (getChurch, updateChurch, members) |
| `hymns.api.js` | Fonctions d'appel (getHymns, favorites, playlists) |
| `bible.api.js` | Fonctions d'appel (getBooks, getVerse, search) |
| `noise.api.js` | Fonctions d'appel (submitReading, getStats, alerts) |

### **3. Contextes (context/)**

| Fichier | Contenu |
|---------|---------|
| `AuthContext.jsx` | Gestion de l'authentification (user, token, login, logout) |
| `ChurchContext.jsx` | Gestion de l'église (membres, settings) |
| `SocketContext.jsx` | Gestion Socket.IO (connexion, events) |

### **4. Hooks (hooks/)**

| Fichier | Contenu |
|---------|---------|
| `useAuth.js` | Utiliser AuthContext facilement |
| `useSocket.js` | Utiliser SocketContext facilement |
| `useNoise.js` | Gestion du bruit (lectures, alertes) |
| `useHymns.js` | Gestion des cantiques (favoris, playlists) |

### **5. Pages (pages/)**

| Fichier | Contenu |
|---------|---------|
| `HomePage.jsx` | Page d'accueil publique |
| `DashboardPage.jsx` | Tableau de bord après connexion |
| `BiblePage.jsx` | Lecture et recherche de la Bible |
| `HymnsPage.jsx` | Gestion des cantiques |
| `ServicesPage.jsx` | Gestion des cultes |
| `AnnouncementsPage.jsx` | Gestion des annonces |
| `ChurchPage.jsx` | Gestion de l'église |
| `NoisePage.jsx` | Moniteur de bruit |
| `SettingsPage.jsx` | Paramètres utilisateur |

### **6. Composants (components/)**

#### **Common**
- `Layout.jsx` : Structure principale (sidebar + content)
- `Header.jsx` : Barre de navigation supérieure
- `Footer.jsx` : Pied de page
- `Sidebar.jsx` : Menu latéral
- `Loading.jsx` : Spinner de chargement

#### **Auth**
- `LoginPage.jsx` : Connexion par téléphone
- `OTPVerification.jsx` : Validation du code OTP
- `ChooseChurch.jsx` : Sélection de l'église

#### **Dashboard**
- `Dashboard.jsx` : Vue principale
- `StatsCards.jsx` : Statistiques
- `RecentActivity.jsx` : Activités récentes

#### **Bible**
- `BibleReader.jsx` : Lecture des versets
- `BibleSearch.jsx` : Recherche avancée
- `VerseOfTheDay.jsx` : Verset du jour

#### **Hymns**
- `HymnList.jsx` : Liste des cantiques
- `HymnViewer.jsx` : Visualisation des paroles
- `HymnSearch.jsx` : Recherche de cantiques
- `HymnFavorites.jsx` : Favoris
- `HymnPlaylist.jsx` : Playlists

#### **Services**
- `ServiceList.jsx` : Liste des cultes
- `ServiceOrder.jsx` : Ordre du culte
- `ServiceBuilder.jsx` : Construction du culte (drag & drop)

#### **Noise**
- `NoiseMonitor.jsx` : Moniteur en temps réel
- `NoiseChart.jsx` : Graphique d'historique
- `NoiseAlerts.jsx` : Alertes
- `NoiseSettings.jsx` : Paramètres

### **7. Utilitaires (utils/)**

| Fichier | Contenu |
|---------|---------|
| `constants.js` | Constantes globales (rôles, statuts) |
| `formatters.js` | Formateurs (date, nombre, etc.) |
| `helpers.js` | Fonctions utilitaires |
